Methods for handling authentication.

Methods for handling authentication. Usage of this module may not be needed. When making calls to the API using other modules in this sdk, authorization is handeled for you. This module is available if you would like to use it to make standard restful calls to the API outside of the SDK.

Methods

token() → {Promise.<Token>}

Creates a new token from the Fortuna API Authorization server and stores it on disk. Calling the method will return a promise containing the token. Subsequent calls will check the freshness of the token and create a new one if needed.
